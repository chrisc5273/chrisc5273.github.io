Module 1: The CIA Triad

The foundation of information security.

  • Confidentiality: Preventing unauthorized disclosure of information.
  • Integrity: Ensuring information is not altered or destroyed in an unauthorized manner.
  • Availability: Ensuring information and systems are accessible when needed.

Module 2: Access Control

Mechanisms for controlling who can access what.

  • Authentication: Verifying a user's identity (e.g., password, 2FA).
  • Authorization: Granting specific permissions to an authenticated user.
  • Accountability: Tracking user actions.

Module 3: Networking Basics (OSI Model)

A conceptual framework for how networking systems communicate.

  1. Physical
  2. Data Link (e.g., MAC Addresses)
  3. Network (e.g., IP Addresses, Routers)
  4. Transport (e.g., TCP/UDP, Ports)
  5. Session
  6. Presentation
  7. Application (e.g., HTTP, FTP)
